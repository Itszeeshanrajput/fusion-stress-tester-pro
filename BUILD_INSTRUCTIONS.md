# Building Fusion Stress Tester PRO Executable

## Quick Build

### Option 1: Using Build Script (Recommended)

```bash
python build_exe.py
```

This will create a single-file executable in the `dist/` folder.

---

### Option 2: Manual PyInstaller Command

```bash
pyinstaller --name=FusionStressTesterPRO --onefile --windowed fusion_stress_tester_pro_v0.0.9.py --collect-all=customtkinter --collect-all=matplotlib --hidden-import=psutil --hidden-import=pynvml --hidden-import=torch
```

---

## Build Output

After building, you'll find:
- **Executable:** `dist/FusionStressTesterPRO.exe`
- **Build files:** `build/` folder (can be deleted)
- **Spec file:** `FusionStressTesterPRO.spec` (can be kept for future builds)

## Creating Release Package

### Step 1: Build the Executable

```bash
python build_exe.py
```

### Step 2: Create Release Folder

```bash
mkdir release
mkdir release\FusionStressTesterPRO_v1.0.0
```

### Step 3: Copy Files

Copy these files to the release folder:
- `dist\FusionStressTesterPRO.exe`
- `README.md`
- `LICENSE`
- `docs\USER_GUIDE.md`
- `CHANGELOG.md`
- `RELEASE_NOTES_v1.0.0.md`

### Step 4: Create ZIP

```bash
cd release
tar -a -c -f FusionStressTesterPRO_v1.0.0_Windows.zip FusionStressTesterPRO_v1.0.0
```

Or use Windows Explorer to create a ZIP file.

## Troubleshooting

### Issue: "Failed to execute script"

**Solution:** Make sure all dependencies are installed:
```bash
pip install -r requirements.txt
pip install pyinstaller
```

### Issue: Missing DLL errors

**Solution:** Install Visual C++ Redistributable:
- Download from Microsoft: https://aka.ms/vs/17/release/vc_redist.x64.exe

### Issue: Large file size

**Solution:** This is normal. The executable includes:
- Python interpreter
- All libraries (CustomTkinter, Matplotlib, etc.)
- Expected size: 100-200 MB

### Issue: Antivirus flags the executable

**Solution:** This is a false positive. You can:
- Add exception in antivirus
- Build with `--debug` flag to see what's happening
- Sign the executable (requires code signing certificate)

## Advanced Options

### Custom Icon

1. Create or download an `.ico` file
2. Save as `icon.ico` in project folder
3. Update build command:
   ```bash
   pyinstaller --icon=icon.ico ...
   ```

### Debug Build

For troubleshooting, create a console version:
```bash
pyinstaller --name=FusionStressTesterPRO_Debug --onefile fusion_stress_tester_pro_v0.0.9.py --collect-all=customtkinter
```

This will show console output for debugging.

### Optimize Size

To reduce file size:
```bash
pyinstaller --name=FusionStressTesterPRO --onefile --windowed --exclude-module=torch fusion_stress_tester_pro_v0.0.9.py
```

Note: This excludes GPU stress testing.
